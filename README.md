# Ctrl Markdown 编辑器

一个简洁美观的本地 Markdown 编辑器，基于 Electron 构建，支持实时预览、自动保存和快捷键操作。

## 功能特性

### 📁 文件管理
- **打开文件夹**：加载文件夹中的所有 `.md` 文件
- **新建文件**：在打开的文件夹中创建新的 Markdown 文件
- **保存文件**：手动保存或自动保存当前文件
- **删除文件**：安全删除选中的 Markdown 文件

### ✍️ 编辑功能
- **实时编辑**：干净简洁的编辑界面
- **语法高亮**：基本的 Markdown 语法预览
- **快捷键支持**：`Ctrl+S` 快速保存
- **自动保存**：每10秒自动保存，防止内容丢失

### 👁️ 预览功能
- **实时预览**：在编辑和预览模式之间切换
- **完整的 Markdown 支持**：
  - 标题（H1-H6）
  - 加粗、斜体、删除线
  - 代码块（行内和块级）
  - 有序/无序列表
  - 表格
  - 链接和图片
  - 水平线
  - 引用块

### 🎨 界面特性
- **自定义标题栏**：macOS 风格窗口控制（红黄绿按钮）
- **深色主题**：舒适的暗色界面
- **响应式设计**：自适应窗口大小
- **文件列表**：左侧清晰的文件导航

## 安装与使用

### 安装依赖
```bash
npm install
```

### 启动应用
```bash
npm start
```

### 打包应用
```bash
npm run package
```

## 使用方法

1. **打开文件夹**：点击左上角文件夹图标
2. **选择文件**：在左侧文件列表中点击文件
3. **编辑内容**：在右侧编辑区编写 Markdown
4. **预览**：点击预览按钮查看渲染效果
5. **保存**：点击保存按钮或使用 `Ctrl+S`
6. **新建文件**：点击新建按钮，输入文件名
7. **删除文件**：点击删除按钮，确认删除

## 技术栈

- **前端**：原生 JavaScript + HTML5 + CSS3
- **框架**：Electron 41.1.1
- **构建工具**：electron-builder
- **样式**：自定义 CSS，支持暗色主题

## API 接口

### 主进程 API (`main.js`)
- `dialog:openFolder` - 打开文件夹对话框
- `file:readMDFiles` - 读取文件夹中的 Markdown 文件
- `file:saveFile` - 保存文件
- `dialog:saveAs` - 另存为对话框
- `file:deleteFile` - 删除文件
- `file:createFile` - 创建新文件
- 窗口控制：最小化、最大化、关闭

### 预加载脚本 (`preload.js`)
暴露 `window.electronAPI` 对象，包含所有文件操作和窗口控制方法。

## 支持的 Markdown 语法

| 语法 | 示例 | 说明 |
|------|------|------|
| 标题 | `# 标题` `## 二级标题` | 支持 H1-H6 |
| 加粗 | `**粗体**` 或 `__粗体__` | |
| 斜体 | `*斜体*` 或 `_斜体_` | |
| 删除线 | `~~删除线~~` | |
| 代码块 | \`\`\`语言\n代码\n\`\`\` | 支持语法高亮 |
| 行内代码 | \`code\` | |
| 链接 | `[文本](https://example.com)` | |
| 图片 | `![alt](url "title")` | |
| 列表 | `- 无序` 或 `1. 有序` | |
| 表格 | `\| 列1 \| 列2 \|` | 基础表格支持 |
| 水平线 | `---` 或 `***` | |
| 引用 | `> 引用内容` | |

## 已知问题与限制

1. **大文件性能**：超大 Markdown 文件可能影响预览性能
2. **复杂表格**：仅支持基础表格格式
3. **图片预览**：需要网络图片或本地相对路径
4. **数学公式**：暂不支持 LaTeX 数学公式
